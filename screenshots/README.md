# Screenshots

Place your screenshots in this folder and reference them in the main README.md.

## Recommended Screenshots

1. **display-end-date-interface.png** - Screenshot showing the Display End Date interface in Directus admin panel
2. **display-end-date-warning.png** - Screenshot showing the warning when an expired date is selected
3. **field-configuration.png** - Screenshot showing how to configure the field in Directus settings

## How to Take Screenshots

1. Open Directus Admin Panel
2. Navigate to your collection with the Display End Date field
3. Create or edit an item
4. Take screenshot of the date picker interface
5. Save as PNG or JPG format
6. Name files descriptively (e.g., `display-end-date-interface.png`)
